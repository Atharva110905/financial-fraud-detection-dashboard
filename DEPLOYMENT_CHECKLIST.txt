TROUBLESHOOTING:
────────────────────────────────────────────────────────────────────────────────

Problem: "Deployment failed" / "App is not running"
→ Check Streamlit Cloud logs (click "Manage app" → "Logs")
→ Verify fraud_detection.db and artifacts/ are in your repo
→ Run locally first: python -m streamlit run app.py

Problem: "Module not found" errors
→ Check requirements.txt has all dependencies
→ Make sure tensorflow-cpu is NOT in requirements.txt

Problem: "First load takes too long"
→ Normal! Streamlit caches data on first load. Subsequent loads are fast.
→ Try reducing transaction count in app if needed
